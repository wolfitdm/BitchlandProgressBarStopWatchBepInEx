BitchlandProgressBarStopWatchBepInEx

Shows a stop watch (progress bar) in the BepInEx Console (in order to see progress in the loading screen)

We all know the problem of getting stuck on the loading screen (without a progress indicator).

We don't know what time it is or how much longer it will take to load.

It's really annoying.

Now we can change that.

We can measure the loading time ourselves and know exactly how long we'll have to wait in the future.

I have to wait 0.47 seconds until everything is loaded.
